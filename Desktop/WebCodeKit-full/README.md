# WebCodeKit — مشروع مدمج (موقع + سيرفر AI)

دلوقتي الموقع والسيرفر بقوا حاجة واحدة. سيرفر Node.js واحد بيعمل اتنين:

1. **يستضيف الموقع نفسه** (كل ملفات HTML/CSS/JS بتاعة WebCodeKit)
2. **يشغّل المساعد الذكي بالـ API الحقيقي** (Claude) بدل القوالب الثابتة،
   مع إخفاء مفتاحك بأمان

مش محتاج تدير سيرفرين. أمر واحد يشغّل كل حاجة.

---

## 1) التثبيت (مرة واحدة بس)

افتح Terminal جوة مجلد المشروع واكتب:

```
npm install
```

## 2) إضافة مفتاح Anthropic

1. اعمل نسخة من `.env.example` واسمها `.env`
2. افتح `.env` وحط مفتاحك الحقيقي:

```
ANTHROPIC_API_KEY=sk-ant-مفتاحك-الحقيقي-هنا
PORT=3000
ALLOWED_ORIGIN=http://localhost:3000
```

⚠️ **ملف `.env` ميترفعش على GitHub أبداً** — فيه `.gitignore` جاهز يتجاهله تلقائياً.

## 3) التشغيل

```
npm start
```

هتشوف:
```
✅ السيرفر شغال على http://localhost:3000
🌐 الموقع نفسه: http://localhost:3000/index.html
🤖 المساعد الذكي: http://localhost:3000/smart-ai.html
```

افتح الرابط `http://localhost:3000` في المتصفح — هتلاقي الموقع كامل،
وصفحة `smart-ai.html` فيها المساعد شغال بالذكاء الحقيقي مباشرة، من غير
أي تعديل تاني.

---

## إزاي شغالة بالضبط؟

- قبل كنت عندك ملفات الموقع في مكان، والسيرفر في مكان تاني، وكل واحد
  بيتشغّل لوحده وبيتكلموا عبر رابط (`http://localhost:3000`).
- دلوقتي السيرفر (`server.js`) بقى يستخدم `express.static` عشان "يقدّم"
  ملفات الموقع للمتصفح مباشرة (بدل ما تحتاج Live Server أو سيرفر تاني للموقع).
- في نفس الوقت، أي طلب يجي على `/api/chat` أو `/api/code` بيتعامل معاه
  السيرفر بنفسه، يضيف مفتاح Anthropic من `.env`، ويرجّع الرد.
- بما إن الموقع والـ API على نفس الدومين دلوقتي، ملف `smart-ai.js` بقى
  يستخدم مسار نسبي (`/api/chat`) بدل رابط كامل — يعني لو نشرت السيرفر
  على دومين تاني، هيفضل شغال زي ما هو من غير أي تعديل.

## النشر (Deployment)

أي استضافة بتشغّل Node.js تنفع (Render, Railway, Fly.io, VPS عادي):

1. ارفع المشروع كامل (غير `.env` و `node_modules`)
2. حدّد متغيرات البيئة من لوحة تحكم الاستضافة: `ANTHROPIC_API_KEY`, `PORT`, `ALLOWED_ORIGIN`
3. أمر التشغيل: `npm start`
4. بعد النشر، حدّث `ALLOWED_ORIGIN` ليكون دومينك الحقيقي (مش `*`) لمنع
   استخدام مفتاحك من مواقع تانية.

## لو السيرفر مش شغال؟

الموقع مايتوقفش — `smart-ai.js` فيه نظام fallback تلقائي: لو السيرفر
مش رادّ لأي سبب، المساعد يرجع تلقائياً للقوالب الجاهزة (Demo mode) عشان
المستخدم يفضل شايف رد بدل ما الصفحة تتعطل.

## الـ Endpoints

| Endpoint | الوظيفة |
|---|---|
| `GET /api/health` | فحص إن السيرفر شغال |
| `POST /api/chat` | أسئلة عادية (شرح، نصايح) |
| `POST /api/code` | طلبات توليد كود (HTML/CSS/JS) |
| باقي المسارات | ملفات الموقع (HTML/CSS/JS) |

كل الـ API محمي بحد 30 طلب/دقيقة لكل IP.
